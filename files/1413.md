# 1413. Minimum Value to Get Positive Step by Step Sum

```java
class Solution {
    public int minStartValue(int[] nums) {
        int res = 0;
        for(int i=0;i<nums.length;i++) {
            if(i!=0) {
                nums[i] += nums[i-1];
            }
            //System.out.print(nums[i]+" ");
            //[-3,2,-3,4,2]
            //-3, -1,-4,0,2
            //4, 2, 5, 1, -1
            res = Math.max(1, Math.max(res, 1-nums[i]) );
        }
        
        return res;
    }
}
```
