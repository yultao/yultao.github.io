# 2389. Longest Subsequence With Limited Sum
[LeetCode](https://leetcode.com/problems/longest-subsequence-with-limited-sum/description/)

注意binary search的边界
```java
class Solution {
    public int[] answerQueries(int[] nums, int[] queries) {
        //step 1: sort
        Arrays.sort(nums);
        //System.out.println(Arrays.toString(nums));

        //step 2: prefix sum
        int[] prefixSum = new int[nums.length];
        prefixSum[0] = nums[0];
        for(int i=1; i<nums.length; i++) {
            prefixSum[i] = prefixSum[i-1] + nums[i];
        }
        //System.out.println(Arrays.toString(prefixSum));

        //step 3: binary serch for each query
        int[] res = new int[queries.length];
        
        for(int i=0; i<queries.length; i++) {
            int left = 0;
            int right = prefixSum.length - 1;
            while(left <= right) {

                int mid = left + (right-left)/2;
                //System.out.println("No. "+i+": "+left +", "+ right+", "+ mid);
                //1, 2, 4, 5  nums
                //1, 3, 7, 12 prefixSum
                //   ^
                //   |
                if(prefixSum[mid] <= queries[i]) {
                    res[i] = mid+1;
                    left = mid+1;
                } else {
                    right = mid-1;
                }
                //System.out.println("No. "+i+": "+left +", "+ right+", "+ Arrays.toString(res));
            }
        }
        return res;
    }
}
```
